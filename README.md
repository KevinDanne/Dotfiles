# Dotfiles

## Getting Started
1. Clone the repository to ~/.dotfiles:

```bash
git clone https://github.com/KevinDanne/Dotfiles.git ~/.dotfiles
```

2. Execute the install script for your distribution

    2.1 Arch linux:
    ```bash
    ./install_arch.sh
    ```
    
    2.2 Ubuntu based distributions:
    ```bash
    ./install_ubuntu.sh
    ```
